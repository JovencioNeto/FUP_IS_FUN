<h1 align="center">FUP – Exercícios Resolvidos em C</h1>

<p align="center">
Repositório contendo soluções de exercícios da disciplina de <strong>Fundamentos de Programação (FUP)</strong>.
</p>

<h2>Sobre o repositório</h2>

<p>
Este repositório reúne exercícios resolvidos durante as atividades da disciplina
<strong>Fundamentos de Programação (FUP)</strong>, utilizando a linguagem C.
As questões foram propostas na plataforma <strong>FUP is Fun</strong>, disponível no Moodle.
</p>

<p>
O objetivo deste repositório é registrar e demonstrar as soluções desenvolvidas
ao longo da disciplina, evidenciando a aplicação prática de conceitos fundamentais
de lógica de programação e programação estruturada.
</p>
